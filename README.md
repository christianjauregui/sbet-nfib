# sbet-nfib
Small Business Economic Trends (SBET) module 

* Creates a database from NFIB's SBET REST API, enabling the collection of microdata and construction of indicators from the monthly survey.
* SBET produces the highly regarded Small Business Optimism Index quoted throughout the financial press every month.
* This code enables the construction of similar indices (as well as microdata distributions) by industry, region, and employee size categories.
